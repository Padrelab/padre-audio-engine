# test-sd-mpr121

Пример для ESP32: проигрывание одного потока с SD-карты из `/music` в случайном порядке (wav/mp3/flac) на I2S, управление тапами MPR121.

## Управление (MPR121)

- Сенсор `0`: пауза/возобновление
- Сенсор `1`: следующий трек
- Сенсор `2`: громкость -1
- Сенсор `3`: громкость +1

## Пины

- SD: `CS=10`, `SCK=12`, `MISO=13`, `MOSI=11`
- I2S: `BCLK=14`, `LRC=15`, `DOUT=16`
- MPR121: `SDA=35`, `SCL=36`, `IRQ=37`

## Порог MPR121

В `test-sd-mpr121.ino`:

- `TOUCH_THRESHOLD`
- `RELEASE_THRESHOLD`

## Структура папки

- `test-sd-mpr121.ino` — основной скетч
- `patches/RandomPlaylist.*` — сканирование `/music` + shuffle
- `patches/Mpr121TapInput.*` — инициализация MPR121 и детектор тапов
- `platformio.ini` — зависимости (можно собрать папку как отдельный PlatformIO-проект)

## Быстрый запуск

1. Положить аудиофайлы (`.wav`, `.mp3`, `.flac`) на SD в папку `/music`.
2. Открыть эту папку как PlatformIO проект.
3. Залить на ESP32.
